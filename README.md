# Breakable Tiles
These (incredibly epic) scripts allow you to add custom, breakable tiles to Unity 2D


# How to use
Steps:
### 1. Create an empty game object and assign it the `TilemapManager.cs` script, then assign your tilemap to it.
>
> ![image](https://github.com/5KFubi/Unity2D-Tiles-Wtih-HP/assets/114242961/72c1ecd8-961d-4998-8c24-a0e1bbf29c04)

### 2. Assets > Right-Click > Create > Tiles, then create a `Custom Tile` and a `Custom Tile Data`.
>
> ![image](https://github.com/5KFubi/Unity2D-Tiles-Wtih-HP/assets/114242961/93e1e123-4a2e-451e-b5a6-1359dbb10b84)

### 3. Setup your desired tile HP in the `Custom Tile Data` you just made (default is 3).
>
> ![image](https://github.com/5KFubi/Unity2D-Tiles-Wtih-HP/assets/114242961/4ed7971e-60e6-4b07-a4b0-a3b8a4f41ef0)

### 4. Insert the desired `Custom Tile Data` into the `Custom Tile`.
> [Optional] Create a rule tile and insert it into the Custom Tile.
>
> [Optional] Set the **Sprite** to something to see it when placing into the grid.
>
> ![image](https://github.com/5KFubi/Unity2D-Tiles-Wtih-HP/assets/114242961/01aa74b0-9b6f-4c09-8973-0eb29ec743ab)
